📦 WHAT YOU NEED TO DO RIGHT NOW (5 minutes)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1️⃣  CREATE .env FILE
   
   Run this:
   ┌────────────────────────────────────────────────────────────────┐
   │ ./setup.sh                                                     │
   └────────────────────────────────────────────────────────────────┘
   
   OR manually:
   ┌────────────────────────────────────────────────────────────────┐
   │ cp .env.example .env                                           │
   │ # Then edit .env and add your OpenAI API key                   │
   └────────────────────────────────────────────────────────────────┘

2️⃣  GET OPENAI API KEY (Required for AI features)
   
   → Go to: https://platform.openai.com/api-keys
   → Click "Create new secret key"
   → Copy key (starts with sk-)
   → Add to .env file:
     OPENAI_API_KEY=sk-your-actual-key-here

3️⃣  DEPLOY FIRESTORE RULES (CRITICAL!)
   
   → Go to: https://console.firebase.google.com
   → Select: collegeclimb-ai
   → Go to: Firestore Database → Rules
   → Copy ALL 237 lines from: firestore.rules
   → Paste and click: Publish

4️⃣  START THE SERVER
   
   ┌────────────────────────────────────────────────────────────────┐
   │ npm run dev                                                    │
   │ # Open http://localhost:3000                                   │
   └────────────────────────────────────────────────────────────────┘

1. Add environment variables in Vercel:
   vercel env add OPENAI_API_KEY
   vercel env add COLLEGE_SCORECARD_API_KEY

2. Deploy:
   vercel --prod

3. Add domain to Firebase Auth:
   → Firebase Console → Authentication → Settings
   → Add authorized domain: your-app.vercel.app
